# Настройка блока

## Блок почтовых сообщений

### Добавить блок

На странице настройки нажмите кнопку **Создать блок** и выберите блок **Таблица почты**, чтобы добавить блок почтовых сообщений.

![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_09_56_PM.png)

### Настройка полей

Нажмите кнопку **Поля** у блока, чтобы выбрать поля, которые нужно отображать. Подробные операции см. в настройке полей таблицы.

![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_09_58_PM.png)

### Установить область данных

В правой панели настроек блока можно выбрать область данных: все письма или письма текущего вошедшего пользователя.

![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_09_58_PM%20(1).png)

### Фильтрация данных по почтовому адресу

Нажмите кнопку настройки справа у блока почтовых сообщений и выберите **Область данных**, чтобы задать диапазон данных для фильтрации писем.

![](https://static-docs.nocobase.com/email-manager/Emails-12-02-2025_04_32_PM.png)

Настройте условия фильтра, выберите поле почтового адреса, по которому нужно фильтровать, и нажмите **ОК**, чтобы сохранить.

![](https://static-docs.nocobase.com/email-manager/Emails-12-02-2025_04_26_PM.png)

Блок почтовых сообщений будет показывать письма, соответствующие условиям фильтра.

![](https://static-docs.nocobase.com/email-manager/Emails-12-02-2025_04_29_PM.png)

> Фильтрация по почтовому адресу не чувствительна к регистру.

### Фильтрация данных по почтовому домену

Создайте поле в бизнес-таблице для хранения информации о почтовом домене (тип JSON), чтобы затем использовать его для фильтрации почтовых сообщений.

![](https://static-docs.nocobase.com/email-manager/data-source-manager-main-NocoBase-12-02-2025_04_36_PM.png)

Заполните информацию о почтовом домене.

![](https://static-docs.nocobase.com/email-manager/Emails-12-02-2025_04_38_PM.png)

Нажмите кнопку настройки справа у блока почтовых сообщений и выберите **Область данных**, чтобы задать диапазон данных для фильтрации писем.

![](https://static-docs.nocobase.com/email-manager/Emails-12-02-2025_04_32_PM.png)

Настройте условия фильтра, выберите поле почтового домена, по которому нужно фильтровать, и нажмите **ОК**, чтобы сохранить.

![](https://static-docs.nocobase.com/email-manager/Emails-12-02-2025_04_41_PM.png)

Таблица почтовых сообщений будет показывать письма, соответствующие условиям фильтра.

![](https://static-docs.nocobase.com/email-manager/Emails-12-02-2025_04_48_PM.png)

## Блок деталей почты

Сначала включите функцию **Включить открытие по клику** в полях блока почтовых сообщений.

![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_10_01_PM.png)

Добавьте блок **Детали почты** во всплывающее окно.

![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_10_02_PM.png)

Теперь вы можете просматривать подробное содержимое письма.

![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_10_03_PM.png)

Вы можете настроить нужные кнопки внизу.

> Если текущее письмо находится в статусе черновика, по умолчанию отображается форма редактирования черновика.

## Блок отправки почты

Есть два способа создать форму отправки почты:

1. Добавить кнопку **Отправить письмо** сверху таблицы:  
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_04_PM.png)

2. Добавить блок **Отправка почты**:  
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_05_PM.png)

Оба способа создают полноценную форму отправки почты.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_05_PM%20(1).png)

Каждое поле формы почты соответствует обычным формам и может настраиваться, например, с **значениями по умолчанию** или **правилами связывания** и т. п.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_03_58_PM.png)
